# SupplyChain Sentinel AI

An autonomous multi-agent supply chain intelligence system that detects disruptions, analyzes global risk signals, and recommends optimal suppliers and logistics actions in real time using coordinated AI agents.

## 🚀 Architecture

The system demonstrates a robust multi-agent workflow orchestrated sequentially using Next.js 15, AI/ML APIs, and Featherless AI.

### Core Tech Stack
- **Frontend**: Next.js 15 (App Router), TypeScript, Tailwind CSS, Shadcn UI, Framer Motion
- **Backend Orchestration**: Next.js Serverless API routes (simulating Band Orchestration Flow)
- **AI Services**:
  - `AI/ML API`: For risk evaluation and supplier scoring
  - `Featherless AI`: For strategic reasoning and natural language strategy generation

### Agent Pipeline (Band Workflow)
1. **Signal Monitoring Agent**: Detects anomalies and demand spikes globally.
2. **Disruption Detection Agent**: Identifies the risks, assesses the severity, and flags the affected industries.
3. **Alternative Supplier Agent**: Sources alternative suppliers mapped by product type and region.
4. **AI/ML Risk Scoring Agent**: Evaluates supplier trust, reliability, and associated risk levels.
5. **Strategy Agent (Featherless AI)**: Processes all signals to formulate a final actionable logistics recommendation.

## ⚙️ Getting Started

### Prerequisites
- Node.js (v20+)
- Valid API keys for Featherless and AI/ML API.

### Installation

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables in a `.env` file:
```env
BAND_API_KEY=your_band_api_key
FEATHERLESS_API_KEY=your_featherless_api_key
AIML_API_KEY=your_aiml_api_key
```

3. Run the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3006`.

### Usage
Enter a prompt such as **"There is a cement shortage in Botswana"** to trigger the autonomous multi-agent pipeline and witness the strategic response generated by the system.
